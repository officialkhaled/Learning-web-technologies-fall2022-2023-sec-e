------------------------
-- Keyboard Shortcuts --
------------------------
Shift + Alt + A : Comment out selected section/part of code (Same for Undo)
Alt + Shift + Down Arrow : To duplicate the current line
Ctrl + B : To open/close the left menu tab
Alt + Click : To create multiple cursors and select multiple lines and type in same code 
Alt + Arrow(up/down) : To move a line up/down
Ctrl + D : To select the same word from multiple lines and create multiple cursors (replace)
Ctrl + I : HTML Tag Wrapper
